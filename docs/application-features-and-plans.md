# 書籍検索アプリ 紹介ドキュメント

このドキュメントは、書籍検索アプリケーションの機能とユーザープランについて説明します。
このアプリケーションは、ユーザーがお気に入りの書籍を見つけ、読書体験を豊かにするためのプラットフォームです。

## ユーザープランと機能一覧

ユーザーは3つのプランから選択できます。プランによって利用できる機能が異なります。

| 機能                  | ゲスト (未ログイン) | スタンダード (無料会員) | プレミアム (有料会員) |
| :-------------------- | :-----------------: | :--------------------: | :--------------------: |
| **読書範囲**          |     試し読みのみ    |       試し読みのみ     |         読み放題       |
| **書籍の検索**        |          ✅         |           ✅           |           ✅           |
| **レビュー閲覧**      |          ❌         |           ✅           |           ✅           |
| **レビュー投稿**      |          ❌         |           ❌           |           ✅           |
| **お気に入り登録**    |          ❌         |           ✅           |           ✅           |
| **ブックマーク機能**  |          ❌         |           ❌           |           ✅           |
| **テーマの設定**      |          ❌         |           ✅           |           ✅           |

---

## 各プランの詳細

### 1. ゲスト (Guest)

ログインしていないユーザー向けのプランです。基本的な機能をお試しいただけます。

-   **説明**: GUEST (未ログイン)
-   **価格**: -
-   **利用可能な機能**:
    -   書籍の試し読み
    -   書籍の検索

### 2. スタンダード (Standard)

無料で会員登録することで利用できるプランです。書籍の検索やレビューの閲覧など、より多くの機能が利用できます。

-   **説明**: STANDARD (無料会員)
-   **価格**: ¥0 / 月
-   **利用可能な機能**:
    -   書籍の試し読み
    -   書籍の検索
    -   レビューの閲覧
    -   お気に入り登録
    -   テーマの設定

### 3. プレミアム (Premium)

月額料金で全ての機能を利用できるプランです。試し読みだけでなく、対象の書籍が読み放題になります。

-   **説明**: PREMIUM (有料会員)
-   **価格**: ¥980 / 月
-   **利用可能な機能**:
    -   **全ての書籍が読み放題**
    -   書籍の検索
    -   レビューの閲覧
    -   レビューの投稿
    -   お気に入り登録
    -   ブックマーク機能
    -   テーマの設定
